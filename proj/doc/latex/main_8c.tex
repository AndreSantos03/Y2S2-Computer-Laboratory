{\footnotesize\ttfamily int handle\+\_\+interrupts (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Handles all game interrupts. 

This function initializes the video mode, loads sprites, sets the timer frequency, and subscribes to various interrupts (keyboard, mouse, timer, RTC). It also contains the main game loop, which handles drawing the game screen and responding to hardware interrupts. At the end, it unsubscribes from interrupts and cleans up resources.

\begin{DoxyReturn}{Returns}
int 0 on success, 1 on failure.

{\footnotesize\ttfamily int main (\begin{DoxyParamCaption}\item[{int}]{argc,  }\item[{char $\ast$$\ast$}]{argv }\end{DoxyParamCaption})}



Main entry point of the game. 

This function sets up the language for LCF messages, enables tracing of function invocations, redirects printf outputs to a file, and starts the LCF control flow. Finally, it performs cleanup tasks before exiting.
